Sitios Web

Son sitios que contiene paginas web organizadas jerarquicamente, son espacios virtuales que contienen toda la informacion que una persona o empresa quieres difundir a travez de internet

Diferencias entre un sitio web y una pagina web

Una pagina web es un documento electronico donde podemos encontrar:

  • Texto
  • Video
  • Imagenes
  • Audios
  • Animaciones

Por su parte un sitio web es una coleccion de paginas web relacionadas entre si
