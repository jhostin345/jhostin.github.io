¿Que es una direccion IP y como funiona?

Los protocolos de internet INTERNET PROTOCOL, son las anclas de internet conectan a la red mundial y apuntalan toda su vida digital. De hecho, es imposible conectarse a internet sin ninguna direccion IP

Analisis de las direcciones IP

Suele consistir en cuatro numeros del 0 al 255, separados por puntos. Dentro de una direccion IP, puede ver el ID de red , asignado a su red y el ID del HOST, el identificador unico asignado a cada dispositivo conectado a esa red especifica

Este es un ejemplo de direccion de IP: 192.168.1.0

ID de red

Es la parte de una direccion IP con la que se identifica la red que se esta tilizando para conectarse a internet, es decir los tres primeros numeros

ID DEL HOST (Identificador del Anfitrion)

Una red puede ser tan pequeña como dos ordenadores conectados entre si o tan grande como internet en si mismo, la ID del anfitrion indica el dispositivo concreto que esta usando para conectarse a una red

Ejemplo

192.168.1.3


Los numero de color rojo representan al ID de la red, los numeros de color verde representan el ID del anfitrion
