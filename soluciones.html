Soluciones web

Son parametros que utilizamos para medir la factibilidad de uso de una pagina web

Usabilidad

Se refiere a la factibilidad de uso que tiene una agina o aplicacion web
La Usabilidad mide tres factores:

  • Eficiencia: el usuario satisface la necesidad de su busqueda
  • Eficacia:el usuario logra alcanzar rapìdamente su cometido
  • Satisfaccion:el usuario se sintio agradable al navegar por la pagina web

Porque es importante la usabilidad en una pagina web

La falta de usabilidad web es mayor por la causa de abandono de los usuarios

Interactividad

Un sitio web interactivo es aquel en que los usuarios pueden interactuar con diferentes elementos y tener una experiencia de usuario que no se limita a hacer click y desplazarse. Estos son hechos por diseñadores y desarrolladores cuyo trabajo principal es ofrecer un sentido de personalizacion

¿Cuales son los beneficios de un sitio web interactivo?

  1. Aumentan proporcionalmente la participacion del usuario y ayudan a retener su atencion
  2. Es probable que los usuarios pasen mas tiempo en el sitio web
  3. Los sitios web interactivos rindan una experiencia de usuario personalizada y ayudan a aumentar el numero de clientes satisfechos
  4. Crean un imacto positivo en los usuarios y genera un buen recuerdo
  5. Existen mayor probabilidad que los usuarios recomienden este sitio web a otros

Funciones interactivas

  1. Google maps: Proporciona una buena forma de mejorar la interaccion del usuario y aumenta la confianza
  2. Crear votaciones: Mostar los resultados de las encuestas ayuda aumentar la confianza y el compromiso
  3. Funciones de la barra de busqueda: Dentro del sitio ayudan al usuario al navegar de manera mas efectiva. Funciones de filtrado de ordenacion, entre otras
  4. Calificaciones: Son una forma visual rapìda de obtener comentarios para mejorar los productos y servicios

    5. Elementos esenciales

    A la hora de realizar una pagina web que sea usable debemos tener cuatro elementos:

    identidad

    Es la personalidad que refleja el sitio aqui es importante que la "HOME" describa adecuadamente que es lo que ofrecemos y cual es nuestra actividad

    accesibilidad

    Es la posibilidad de que todos los individuos puedan hacr uso de las webs con independencia de las limitaciones del individuo y del dispositivo

    navegacion

    Entendemos como tal a la facilidad de deplazarse por las distintas paginas que distinguen la web.El usuario debe saber en todo momento en que parte de la web se encuentra i como ir a cada seccion.
