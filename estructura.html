DOCTYPE

Nos indica el tipo de documento con el que estamos trabajando, en este caso html



Html lang

Esta etiqueta nos sirve para definir el lenguaje natural con el que vamos a trabajar,por ejemplo:
"en " define que trabajaremos en ingles





HEAD

Es el encabezado del documento html, donde ubicamos a quello que el usuario no vera directamente pero que es esencial para que funicione el sitio web



MetaEtiquetas

Son elementos que se incorporan en el encabezado y que resultan invisibles para el visitante normal, pero de gran utilidad para navegadores, diseñadores y programadores

Atributo Descripcion Sintaxis
Description Descripcion del contenido
Keywords Pueden usarlas para que el motor de busqueda tenga mas informacion acerca del contenido de la pagina
Charset Indica la codificacion de caracteres

Title

Esta etiqueta nos sirve para darle un titulo general a la pagina que es visible para el usuario, se ubica en la pestaña del navegador



Link

Especifica la relacion entre el documento html y un recuros externo



Body

Se refiero al cuerpo del docuemnto es lo que eol usuario va apoder visualizar
