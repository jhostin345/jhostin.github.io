Motores de busqueda.

Es un conjunto de programas coordinados se encarga de visitar cada uno de los sitios que se integran la web

Empleando los propios impervinculos contenidos en las paginas web para buscar y leer otros sitios

Crear un enorme indice de todas las paginas que han sido leidas y registradas

Presenta direcciones en internet como resultado de las peticiones de busquedas solicitadas

Tipos de motores de busqueda

Robots, aranas, rastreadores

Los motores de busqueda en la web consiste en vincular la tecnologia del seguimiento de llamados robots, aranas, orugas, o agentes que viajan en los sitios a intervalos regulares y de forma automatica para descubrir nuevas direcciones

  • Siguen los hipervinculos en cada pagina

  • Cada pagina se identifica a continuacion, el conenido esta indexado a una base de datos accesible o por el internet para los usuarios que introducen las palabras claves
